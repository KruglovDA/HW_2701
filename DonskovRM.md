# Руководство по GIT
## Локальный репозиторий
### Команды локального репозитория

* git init - создание нового репозитория;
* git add <_file name_> - добавление файла в git; 
* git status - состояние репозитория;
* git commit -m "_Комментарий"_ - фиксирует изменения;
* git log - выводит журнал изменений;
* git branch  <_имя ветки_> - создает новую или переходит на уже существующую
* git branch  -d <_имя ветки_> - удаляет ветку;
* git branch - выводит список всех веток;
* git log --graph - выводит ветки в виде графа;
* git dif - разница в версиях
* git merge <_имя ветки_> - сливает поименованную ветку с текущей;
* git checkout <_имя ветки или коммита_> - позволяет переключиться между версиями или ветками;
* git ignore <_file name_> - указывет git не отслеживать названный файл;

## Удаленный репозиторий
### Команды удаленного репозитория

* git clone <_адрес удаленного репозитория_> - копирует удаленный репозиторий в локальную папку
* git push - отправляет изменения в удаленный репозиторий
* git pull - скачивает текуший репозиторий и автоматически делает merge с нашей версией
